# Job Board Application

A job board web application built following a tutorial.

## Technologies
- Flask
- PostgreSQL
- Mailjet API
- HTML/CSS

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Create `.env` file with your database URL and Mailjet credentials
3. Run: `python app.py`

## Features
- Job listings
- Application form
- Email notifications
- Admin dashboard
